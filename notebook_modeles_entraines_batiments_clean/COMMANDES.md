# Commandes utiles

## Ouvrir le notebook

```bash
jupyter notebook notebooks/notebook_modeles_entraines_batiments.ipynb
```

Ou ouvrir directement ce fichier dans VS Code :

```text
notebooks/notebook_modeles_entraines_batiments.ipynb
```

## Installer les dépendances Python du notebook

```bash
pip install -r requirements_notebook.txt
```
